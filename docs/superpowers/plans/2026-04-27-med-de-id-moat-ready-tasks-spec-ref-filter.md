# med-de-id Moat Ready Tasks Spec Ref Filter Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a read-only `mdid-cli moat ready-tasks --spec-ref SPEC_REF` filter so autonomous controllers can route only currently-ready work tied to a specific spec reference.

**Architecture:** Extend the existing `ready-tasks` inspection surface with the same conjunctive, pre-limit filtering semantics used by `assignments` and `task-graph`. The parser stores an optional exact spec reference, the renderer compares it against each ready node's persisted optional `spec_ref`, and the moat-loop spec documents the read-only contract.

**Tech Stack:** Rust workspace, `mdid-cli`, Cargo integration tests, local JSON moat history fixtures generated by `mdid-cli moat round`.

#[test]
fn cli_filters_ready_tasks_by_exact_spec_ref() {
    let history_path = unique_history_path("ready-tasks-spec-ref");
    let history_path_arg = history_path.to_str().expect("history path should be utf-8");

    let round_output = Command::new(env!("CARGO_BIN_EXE_mdid-cli"))
        .args(["moat", "round", "--history-path", history_path_arg])
        .output()
        .expect("failed to run mdid-cli moat round with history path");
    assert!(round_output.status.success(), "moat round failed: {}", String::from_utf8_lossy(&round_output.stderr));

    let output = Command::new(env!("CARGO_BIN_EXE_mdid-cli"))
        .args([
            "moat",
            "ready-tasks",
            "--history-path",
            history_path_arg,
            "--spec-ref",
            "moat-spec/workflow-audit",
        ])
        .output()
        .expect("failed to run mdid-cli moat ready-tasks with spec-ref filter");

    assert!(output.status.success(), "ready-tasks failed: {}", String::from_utf8_lossy(&output.stderr));
    assert_eq!(
        String::from_utf8_lossy(&output.stdout),
        concat!(
            "moat ready tasks\n",
            "ready_task_entries=1\n",
            "ready_task=planner|spec_planning|spec-workflow-audit|Create spec for workflow audit|moat-spec/workflow-audit\n",
        )
    );
}
